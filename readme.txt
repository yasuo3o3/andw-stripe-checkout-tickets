== Description ==
Stripe Checkout を利用したシンプルなチケット販売とユーザー登録フローを提供します。

== Installation ==
1. プラグイン ZIP をアップロードして有効化します。
2. 管理画面の「andW Tickets」メニューで API キーや URL を設定します。
3. `[andw_ticket_buttons]` や `[andw_mypage]` などのショートコードをページへ挿入します。

== Frequently Asked Questions ==
= テキストドメインは何ですか？ =
`andw-stripe-checkout-tickets` です。翻訳ファイルは `/languages` ディレクトリに配置してください。
